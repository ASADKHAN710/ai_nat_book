# AI-Driven Technical Book

Welcome to the AI-Driven Technical Book with Embedded RAG Chatbot. This book is designed to teach advanced robotics and AI concepts through practical examples and hands-on learning.

## Modules

This book is organized into modules, each focusing on a specific aspect of AI-driven robotics:

- **ROS 2 Nervous System**: Learn how ROS 2 serves as middleware connecting AI logic to humanoid robot bodies.

Each module builds upon the previous ones, providing a comprehensive learning path from basic concepts to advanced implementations.

## About This Book

This technical book uses Docusaurus for documentation and includes an embedded RAG chatbot for contextual Q&A. The content is designed for AI students with basic Python knowledge and no prior robotics experience.
